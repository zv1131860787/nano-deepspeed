nano-deepspeed

一个可运行、可阅读、可对比的 DeepSpeed ZeRO 教学复线项目，聚焦核心数据流和通信机制，而不是生产替代。

查看仓库 看实验结果

一句话简历描述

复线官方 DeepSpeed 的 initialize / init_distributed、ZeRO-0/1/2、FP16 动态 loss scaling 与 BF16 autocast，并完成 Qwen3-0.6B 的多卡对齐实验。

2 卡差值: 9.46% 8 卡差值: 1.08% 50 opt steps

已复线能力

当前范围聚焦 ZeRO 主路径，明确声明不覆盖 ZeRO-3 与 offload。

ZeRO Stage 0/1/2 包含参数/梯度分片与 stage2 reducer 路径。
API 兼容入口 initialize / init_distributed / add_config_arguments。
精度与稳定性 FP16 动态 loss scaling，BF16 autocast 训练路径。
官方对齐脚本 同数据、同模型、同步数，直接对比 nano 与 official 指标。

显存对比快照

Qwen3-0.6B，bf16，opt_step=50，口径为日志中的 cuda_peak_alloc_max_mb。

注: peak_alloc 与 peak_reserved 口径不同，后者更受缓存池策略影响，趋势不一定一致。

快速运行

默认使用本仓库教学实现；官方对比脚本见 README。

torchrun --standalone --nproc_per_node=2 examples/train_qwen3_zero12_nano.py \ --deepspeed_config examples/ds_config_zero2.json \ --model-dtype bfloat16 --seed 42 --steps 50
